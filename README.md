# 🎵 Vortex Audio Player (Аудіо-плеєр)

## 🛠 Стек технологій

Проєкт побудовано на базі найсучаснішого та найшвидшого технологічного стеку:

- **Core:** React 19.2 (Hooks, Functional Components), TypeScript ~6.0, Vite 8.0
- **Routing & State:** React Router 7.16 (Data Loaders для треків, Form Actions, Nested Routes)
- **Backend-as-a-Service:** Supabase Client 2.106 (PostgreSQL, Auth, Storage для зберігання аудіофайлів та обкладинок)
- **Styling & UI-компоненти:**
  - Tailwind CSS v4.3 (використання нового Vite-плагіна `@tailwindcss/vite` для миттєвої збірки стилів)
  - Shadcn UI & Vaul 1.1 (мобільні шторки-дравери та діалоги)
  - Lucide React 1.16 (сучасний пак медіа-іконок)
  - `clsx` & `tailwind-merge` (динамічне злиття класів за допомогою утиліти `cn`)
- **Форми та Валідація:** React Hook Form 7.77 + Zod 4.4 (сувора типізація та валідація даних)
- **Утиліти:** `use-debounce` (оптимізація пошуку), `sonner` (спливаючі сповіщення), `@fontsource-variable/geist` (шрифт Geist)

## 🚀 Ключовий функціонал

- 🔐 **Синхронізована авторизація:** Реєстрація та вхід користувачів (Supabase Auth).
- 🔍 **Мультипараметрична фільтрація:** Розумний `audioLoader` для миттєвої фільтрації треків за жанром, автором чи текстовим пошуковим запитом користувача.
- 🎧 **Інтерактивний плеєр:** Повноцінний кастомний аудіоплеєр із можливістю трекінгу прогресу, регулювання гучності, паузи та циклічного чи випадкового перемикання аудіозаписів.
- 💖 **Система "Обране" (Favorites):** Можливість зберігати улюблені треки в один клік із миттєвою синхронізацією стану профілю в базі даних PostgreSQL.
- 📱 **Адаптивні Медіа-Шторки:** Використання висувних панелей для мобільних пристроїв, що забезпечує зручний нативний UX на смартфонах.

## 🧠 Реалізація та Оптимізація Аудіоплеєра

- **Глобальний стан відтворення:** Логіка HTML5 `Audio` винесена в ізольований контекст/хук. Музика продовжує безперервно грати під час навігації користувача сторінками додатку, зміни фільтрів чи переходу в налаштування профілю.
- **Оптимізація рендерингу:** Компоненти прогрес-бару оптимізовані так, щоб щосекундне оновлення поточного часу треку не викликало перерендеру всього списку аудіозаписів (AudiosList).

## 📱 Особливості UX для Мобільних Пристроїв

- **Native-like Шторки (Vaul):** На екранах смартфонів детальна панель треку, що зараз грає, відкривається знизу вгору за допомогою бібліотеки `Vaul`. Вона підтримує закриття за допомогою свайпу вниз (drag-to-dismiss), як у нативних iOS/Android застосунках.
- **Збереження трафіку:** Текстовий пошук за авторами та треками оптимізовано за допомогою `use-debounce`. Запит до бази даних Supabase відправляється лише через 600 мс після того, як користувач завершив введення, що економить мобільний дата-трафік.

## 🔧 Інсталізація та Локальний Запуск

Щоб запустити цей проєкт локально на своєму комп'ютері, виконайте наступні кроки:

### 1. Клонування репозиторію та перехід у папку

```bash
git clone https://github.com/denyskorniienko22/vortex-audio-player.git
cd vortex-audio-player
```

### 2. Встановлення всіх залежностей

```bash
npm install
```

### 3. Налаштування змінних оточення (.env)

Створіть у корені проєкту файл з назвою .env.local та додайте туди свої ключі з особистого кабінету Supabase:

```bash
VITE_SUPABASE_URL="your_supabase_project_url"
VITE_SUPABASE_ANON_KEY="your_supabase_anon_public_key"
```

### 4. Запуск у режимі розробки

```bash
npm run dev
```
